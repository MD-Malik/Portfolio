# Portfolio 🌇

My Portfolio - Portfolio to showcase skills I have, toolset I use and projects I built along with my personal information.


## Tech Stack Used 👩‍💻

- HTML
- CSS
- Javascript
- React
- Router
- styled-components
- github
- netlify

## Run Locally 💻

Clone the project

```bash
  git clone https://github.com/MD-Malik/Portfolio.git
```

Go to the project directory

```bash
  cd Portfolio
```

Start

```bash
  start ./index.html
```
